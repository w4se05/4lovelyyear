## **Exercises (Template and container classes)** 

1. Implement the linked list as described in Lecturer 12. Operations include: 

   - _append_ : add a node at the end 

   - _prepend_ : add a node at the beginning 

   - _insert_ : insert a node in place 

   - _find_ : find a specific node 

   - _get_ : get a node at the current position 

   - _replace_ : replace the content of a node 

   - _isEmpty_ : find out if the list is empty 

   - _remove_ : remove a node 

   - _clear_ : remove all the nodes 

Test your program. 

2. Design and implement a generic Stack data structure as described in Lec11_ex1 using the linked list. Test your program.
