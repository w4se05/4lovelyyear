## **Exercises** 

**Q1** ) Assume that a set of **Quadrilaterals** consists of **Rectangles** , **Parallelograms** , and **Trapeziums** . Each quadrilateral has the data members of sizes (widths, height) and its area. Write a program that: 

- a. define the Quadrilateral class with a constructor that gives the values to _widths_ and _height (default value: 1)_ . Inside the class defines the _print_ () method that prints the area; 

- b. define three sub-classes Rectangle, Parallelogram, and Trapezium inherited from the Quadrilateral class; write the _double area_ () method to calculate the area of quadrilateral; 

- c. create an array of _N_ quadrilaterals (each element is a rectangle, parallelogram, or trapezium). Set the sizes of each quadrilateral and print its area.
